## Installation

To get started, follow these steps:

1. Clone this repository to your local machine.
```bash
git clone https://github.com/your-username/Dogs-Telegram.git
```

2. Install the necessary dependencies.
```bash
npm install
```

3. Run the application.
```bash
npm start
```
